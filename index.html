Git repository létrehozása

1. lépés

Hozzunk létre egy mappát. Ebből fogunk repository-t csinálni. [user@host] $ mkdir mappa


2. lépés

Navigáljunk a mappába [user@host] $ cd mappa

3. lépés

Ha a mappában vagyunk "indítsuk" a gitet [user@host] $ git init
Initialized empty Git repository in /Users/felhasználó/mappa/.git/ Létrehoztunk egy git repository-t. Készen vagyunk.
